        Arguments:
            offer       List of dicts of items to offer for trade.  See below for

            ask         Integer price in E you're asking for this trade, between 
                        0.1 and 100.

        "offer"

                resources
                    {   'type':         'bauxite',
                        'quantity':     10000   },
                glyphs
                    {   'type':         'glyph',
                        'name':         'bauxite',
                        'quantity':     10000   },
                plans

                        'level':                1,
                        'extra_build_level':    3,                      # If > 1, 'level' must be 1.

                prisoners
                    {   'type':         'prisoner',

                ships
                    {   'type':     'ship',
                        'ship_id':  12345   },

        Returns the ID of the trade just added.
        """
