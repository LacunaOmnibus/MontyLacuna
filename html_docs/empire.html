fetch_captcha()[source]¶

Returns dict containing ‘guid’ and ‘url’ keys DICT not OBJ, so rv[‘guid’], not rv.guid

There also exists a Captcha class, which requires the user to already be logged in. This fetch_captcha method exists to allow a brand new, not-yet-logged-in user to get a captcha, the solution to which they can pass to create().

If you’re looking for a captcha for anything other than new user creation, go use the Captcha class.
