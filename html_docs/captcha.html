Fetches a captcha for the user to solve from the server.

Generally, you don’t need to call this, use showit() instead.

This mirrors the TLE Captcha::fetch() method. It fetches the url and guid for the captcha, but does not actually fetch the captcha image (which lives at url) itself.

Retval includes:
  • guid - a hashed CHECK captcha ID,
  • url - The URL to the image
