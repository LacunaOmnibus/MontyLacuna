exceptions¶

exception lacuna.exceptions.BadBuildingError(value)¶

Bases: builtins.Exception

The building requested exists but isn’t usable (probably damaged)

exception lacuna.exceptions.BadConfigSectionError(section)¶

Bases: builtins.Exception

User specified a config file section which does not exist.

exception lacuna.exceptions.BadCredentialsError(value)¶

Bases: builtins.Exception

One or both of username and/or password is bad. We’re not sure which.

exception lacuna.exceptions.CaptchaResponseError(section)¶

Bases: builtins.Exception

The user’s response to a captcha puzzle was incorrect.

exception lacuna.exceptions.GDIError(value)¶

Bases: builtins.Exception

The client attempted to access own alliance methods, but is not currently in an alliance.

exception lacuna.exceptions.InsufficientResourceError(value)¶

Bases: builtins.Exception

The client has some, but not enough, of some resource. This resource can be any score, eg a building level that’s not high enough.

exception lacuna.exceptions.MissingResourceError(value)¶

Bases: builtins.Exception

Some TLE resource required to do whatever it is you were attempting to do is missing.

exception lacuna.exceptions.NoAvailableShipsError(value)¶

Bases: builtins.Exception

We attempted to do something that requires one or more ships (eg fetch spies), but no ships were available.

exception lacuna.exceptions.NoSuchBuildingError(value)¶

Bases: builtins.Exception

The building requested does not exist

exception lacuna.exceptions.NoSuchEmpireError(value)¶

Bases: builtins.Exception

The empire name used does not exist

exception lacuna.exceptions.NoSuchMyBodyError(value)¶

Bases: builtins.Exception

The body requested does not exist in my empire.

exception lacuna.exceptions.NoSuchShipError(value)¶

Bases: builtins.Exception

The ship name used does not exist

exception lacuna.exceptions.NoUsableSpiesError(value)¶

Bases: builtins.Exception

No spies onsite satisfy the requirements.

exception lacuna.exceptions.NotJsonError(value)¶

Bases: builtins.Exception

The server responded, but handed back something other than JSON

exception lacuna.exceptions.RequestError(value)¶

Bases: builtins.Exception

A network request is not responding properly. Generally this means that response.status_code is other than 200.

exception lacuna.exceptions.ServerError(code: int, message: str)¶

Bases: builtins.Exception

The TLE server returned a 500 and a JSON error string

exception lacuna.exceptions.TopoffError(number: int, message: str)¶

Bases: builtins.Exception

A script got a –topoff argument, but we’re already over the requested number.

exception lacuna.exceptions.WorkingError(value)¶

Bases: builtins.Exception

The user wanted to begin work on some resource (a building, spy, ship, etc), but that resource is already doing something and can’t be assigned a new job until it completes the current one.
