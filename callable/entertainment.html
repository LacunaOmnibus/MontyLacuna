get_lottery_voting_options(*args, **kwargs)¶

Returns a list of external sites that the user can visit. Each visit affords the user a single lottery ticket.

Requires a captcha. If you have not solved one yet with the current session_id, one will be displayed for you.

Returns a dict including key ‘options’, a list of dicts, each dict containing:
  • name – “Some Site”,
  • url – “http://www.somesite.com/vote?id=44” },
