Captcha¶

class lacuna.captcha.Captcha(client: object, *args, **kwargs)¶

Bases: lacuna.bc.LacunaObject

Fetches, displays, and solves graphical captchas.

General usage will be:

cap = my_client.get_captcha()
cap.showit()            # display the captcha image
cap.prompt_user()       # ask the user for a solution
cap.solveit()           # check the user's solution
fetch(**kwargs)¶

Fetches a captcha for the user to solve from the server.

This mirrors the TLE API, but you generally don’t need to call this.

Returns a Puzzle object.

prompt_user()¶

Prompts the user to solve the displayed captcha. It’s not illegal to call this without first calling solveit(), but doing so makes no sense.

showit()¶

Actually downloads the captcha image, and attempts to display it to the user in one of several browsers.

If fetch() is called first, showit() uses that fetched data, but this is not necessary. showit() will call fetch for you.

solve(guid: str, solution: str, **kwargs)¶

Mirrors the TLE Captcha module’s solve() method, but unless you really need this and you really know why, use solveit() instead.

solveit()¶

Sends the user’s response to the server to check for accuracy. Returns True if the user’s response was correct. Raises CaptchaResponseError otherwise.
