accept_from_market(trade_id: int, *args, **kwargs)¶

Accepts a trade from the market. When buying from the SST market, an additional 1 E processing cost is automatically charged in addition to the ‘ask’ price displayed for the trade.

Requires captcha.

Requires ‘trade_id’, integer ID of the trade to purchase.

Get glyphs available to be added to a trade as merchandise.

Returns a tuple:

view_market(page_number: int=1, filter: str='', *args, **kwargs)¶

Shows the trades you currently have offered.

Arguments:
  • page_number – Optional integer page number to view. 25 trades shown per page. Defaults to 1.i
  • filter – Optional string to narrow down the types of trades returned.
    • Valid filters: - energy - food - glyph - ore - plan - prisoner - ship - waste - water
Returns a tuple:
  • trades – List of trading.ExistingTrade objects
  • count – Total number of trades you have up
  • page_number – The page that the “trades” list appeared on. Same value that you passed as an argument, or 1.
