add_to_market(spy_id: int, ask: float, ship_id: int=0, *args, **kwargs)¶

Adds a spy to the mercs market.

Parameters:
  • spy_id -- Required - Integer ID of the spy to add. (-) –
  • ask -- Required - Integer amount of E to charge. Between 1-99 (-) –
  • ship_id -- Optional - Integer ID of the ship to use for the (-) – trade. Obtainable via get_trade_ships(). If not sent, a trade ship will be chosen for you if one is available. This must be the ID of a spy pod.

Returns the integer ID of the trade you just created.

Raises lacuna.exceptions.ServerError 1011 if you do not have any spy pods available.

get_trade_ships(target_body_id: int=0, *args, **kwargs)¶

Returns a list of ships currently onsite that can be used to add a mercenary trade.

Parameters:target_body_id -- Optional integer ID of the body we plan to send (-) – the trade to.

If target_body_id is included, the trade ships returned will include estimated_travel_time, the number of seconds the trip will take. If target_body_id is not passed in, then the estimated_travel_time attribute of the returned objects will be “unknown”.

Returns a list of lacuna.ship.TradeableShip objects.

The only ship type acceptable for adding mercenary trades is a spy pod, so those are the only ships that will be included in the returned list.
